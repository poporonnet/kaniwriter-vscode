# kaniwriter-vscode README

kaniwriter integration for VSCode.

## Features

- Upload the current mruby/c program and open kaniwriter

## Installation

Download the latest `.vsix` file from [release page](https://github.com/poporonnet/kaniwriter-vscode/releases) and install it in VSCode.

## How to use

Open any `.rb` file and click `â–¶` icon in the menu bar.

## Configuration

- `kaniwriter-vscode.compilerUrl`: Change kanicc ( online compiler ) instance URL to use.
- `kaniwriter-vscode.writerUrl`: Change kaniwriter ( web writer ) instance URL to use.
